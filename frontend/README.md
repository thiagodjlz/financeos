# FinanceOS Frontend

Frontend Angular do FinanceOS.

## Requisitos

- Node.js 22.23 ou superior
- Backend local em `http://localhost:8080/api`

## Desenvolvimento

Instalar dependencias:

```bash
npm install
```

Subir a aplicacao:

```bash
npm start
```

Acessar:

```text
http://localhost:4200
```

## Verificacao

Build:

```bash
npm run build
```

Testes:

```bash
npm test -- --watch=false
```
